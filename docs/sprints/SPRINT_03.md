# Sprint 03

## Objetivo

Transformar o Jornada New Job em uma plataforma inteligente através da integração com o Gmail.

Esta Sprint representa o maior diferencial do projeto.

A partir dela o sistema deixará de depender exclusivamente de cadastros manuais.
